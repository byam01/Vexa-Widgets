Vexa

Minimal music.
Minimal interface.

Vexa is a compact music widget designed with a quiet visual presence.
It prioritizes clarity, balance, and subtle interaction so it can Vexa naturally on any screen.

---

Features

• Dynamic album artwork
• Vertical progress indicator
• Dynamic controls
• Device output indicator
• Compact widget layout

---

Design Approach

The goal of Vexa is simple: remove visual noise and leave only what matters.

The interface focuses on a clear visual between artwork, track information, and controls while remaining visually calm regardless of wallpaper or theme.
